ROMstudio

CPS1 arcade studio in the browser.
Play, inspect, modify, and export your personalized version of the game — zero install.

Launch App

Features

Hardware X-Ray

Separate the 4 hardware layers in 3D space. See exactly how the CPS-A/CPS-B ASICs compose each frame — scroll planes, sprites, priority, palette — all in real time while the game runs.

Audio DAW

8 FM channels (YM2151) and 4 ADPCM voices (OKI MSM6295) with VU meters, hit timeline, and mute/solo per channel. Isolate the bass line, melody, or percussion independently.

Piano Roll

Cubase-style Key Editor with vertical keyboard. Select any FM channel and watch the notes scroll in real time — see the music as the hardware plays it.

Sample Browser

Browse, preview, and replace every ADPCM sample in the ROM. Drag a WAV file to swap any sound effect — hear it instantly in-game. Export/import full sample sets as ZIP.

Sprite Editor

Click any sprite on screen to open it in a pixel editor. Paint, flood fill, eyedrop colors — see your changes live in the running game. Undo/redo, neighbor tile navigation.

Palette Viewer

Live grid of all 192 color palettes. Hover for RGB values, click to inspect. Watch palettes animate in real time as the game cycles colors for fire, water, and lighting effects.

Under the Hood

Every component written from scratch in TypeScript. No emulation library — just code and hardware docs.

Main CPU Motorola 68000 Full interpreter, ~3000 lines
Audio CPU Zilog Z80 Autonomous Web Worker
FM Synthesis YM2151 (WASM) Cycle-accurate via Nuked OPM
ADPCM OKI MSM6295 Pure TypeScript decoder
Video CPS-A / CPS-B WebGL2 + Canvas 2D fallback
Audio Output AudioWorklet SharedArrayBuffer ring buffer

~33% CPU on a modern machine — M68K ~25%, Z80+audio ~8%, video ~3%

No ROMs included — provide your own legally obtained MAME ROM sets.

Built with TypeScript + WebGL2 + WASM
