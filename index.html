CPS1 arcade studio in the browser.
Play, inspect, modify, and export your personalized version of the game — zero install.

Launch App

Features

Hardware X-Ray

Separate the 4 hardware layers in 3D space. See exactly how the CPS-A/CPS-B ASICs compose each frame — scroll planes, sprites, priority, palette — all in real time while the game runs.

Live Sprite Editor

Click any sprite on screen to edit it pixel by pixel — see your changes live in the running game. Capture poses, browse animation frames, import images onto any tile. Export your modded ROM as a MAME-compatible ZIP and share it.

Photo Import

Drop any image onto the game background. The engine auto-quantizes it to the CPS1 16-color palette with Atkinson dithering. Your photo becomes part of the game — put your face on an in-game wall.

Palette Swap

Click any color in the live palette grid to change it instantly. Give Guy a pink outfit, turn the sky red, recolor any character or background — the game updates in real time.

Sample Browser

Browse and replace every sound effect in the ROM. Drag a WAV file to swap any ADPCM sample — hear it instantly in-game. Export and import full sample sets as ZIP.

Under the Hood

Every component written from scratch in TypeScript. No emulation library — just code and hardware docs.

Main CPU Motorola 68000 Full interpreter, ~3000 lines
Audio CPU Zilog Z80 Autonomous Web Worker
FM Synthesis YM2151 (WASM) Cycle-accurate via Nuked OPM
ADPCM OKI MSM6295 Pure TypeScript decoder
Video CPS-A / CPS-B WebGL2 + Canvas 2D fallback
Audio Output AudioWorklet SharedArrayBuffer ring buffer

~33% CPU on a modern machine — M68K ~25%, Z80+audio ~8%, video ~3%

No ROMs included — provide your own legally obtained MAME ROM sets.
